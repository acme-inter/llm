Five ways to deploy AI

Cloud APIs — pay per word, no hardware. EU VPS — cheap EU servers but CPU-only. Cloud GPUs — rent by the hour. Owned servers — buy once, low monthly cost. SaaS / Team plans — per-seat ready product.

Why owned servers still have monthly cost

05NVIDIA DGX Spark — deep diveModels, user limits, specs

What it is

The DGX Spark is a palm-sized desktop AI supercomputer — NVIDIA's GB10 Grace Blackwell Superchip, with 128 GB unified memory shared between CPU and GPU. Price: ฿175,000.

It is one physical device. You load one AI model at a time. The model you choose determines how many users can use it simultaneously — that is the only variable you control. Bigger model = smarter but fewer simultaneous users.

The honest limitation

It's a single chip device. The table below shows exactly what happens at each model size: load a 7B model → up to 10 users; load a 14B model → up to 6 users; load a 70B model → only 1 user at a time (the 70B model alone uses 140 GB of the 128 GB — it actually needs Q4 quantisation to fit).

Monthly running cost: electricity only — 170W is very low. At ฿4.5/kWh that is about ฿367/month. Hardware cost ฿175,000 one-time.

KV cache left = 128 GB − model weights. Each user at 4K context needs ~1–2 GB KV cache. Concurrent users = floor(KV cache ÷ 1.5 GB). Token speeds are estimates for NVIDIA Blackwell at INT4/FP16.

06Accuracy: what the score means0–100 explained

How we scored

Blended MMLU (knowledge), HumanEval (coding), MATH (reasoning), MT-Bench (chat) — normalised 0–100. Same model = same score regardless of host.

What the bands mean

90+ Frontier. 80–89 Strong. 70–79 Good (14B). 60–69 Basic (7B). Quantization (Q4) costs ~3 pts. CPU-only → assume low end of band.

DGX Spark accuracy note

09Privacy: where does your data go?Plain language

🟢 HIGH — Self-hosted

Data never leaves your network. Best for medical, legal, finance. Includes DGX Spark, owned servers, cloud GPU instances you control.

🟡 MEDIUM — Enterprise APIs & EU VPS

Data goes to vendor but they do not train on it by contract. IONOS AI API is EU-hosted (GDPR). Claude/OpenAI have DPA available.

🔴 LOW — Consumer / China-hosted

DeepSeek API is hosted in China. Chinese law allows government data access. Free chatbots may train on conversations unless opted out.

📋 Claude Team — specifics

Anthropic does not train on your team's conversations. Encrypted at rest and in transit. BAA available for healthcare. Admin controls.
