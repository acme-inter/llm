Five ways to deploy AI

Cloud APIs — pay per word, no hardware. EU VPS — cheap EU servers but CPU-only. Cloud GPUs — rent by the hour. Owned servers — buy once, low monthly cost. SaaS / Team plans — per-seat ready product.

Why owned servers still have monthly cost

Even buying hardware outright, one real cost remains every month: electricity. A GPU runs 24/7 drawing 170W (DGX Spark) to 1,200W (2× A100). Admin is assumed absorbed by your existing IT team — no extra charge.

What is the DGX Spark?

NVIDIA's DGX Spark (~$3,000 / ฿96,000) is a desktop AI supercomputer with 128 GB unified memory — it can run 70B models in FP16. But it's a single-chip device. Concurrent user capacity is limited by model size. Each card shows exactly how many users it can serve.

05NVIDIA DGX Spark — deep diveModels, user limits, specs

What it is

The DGX Spark is a palm-sized desktop AI supercomputer — NVIDIA's GB10 Grace Blackwell Superchip, with 128 GB unified memory shared between CPU and GPU. Price: ฿175,000.

It is one physical device. You load one AI model at a time. The model you choose determines how many users can use it simultaneously — that is the only variable you control. Bigger model = smarter but fewer simultaneous users.

The honest limitation

It's a single chip device. The table below shows exactly what happens at each model size: load a 7B model → up to 10 users; load a 14B model → up to 6 users; load a 70B model → only 1 user at a time (the 70B model alone uses 140 GB of the 128 GB — it actually needs Q4 quantisation to fit).

Monthly running cost: electricity only — 170W is very low. At ฿4.5/kWh that is about ฿367/month. Hardware cost ฿175,000 one-time.
